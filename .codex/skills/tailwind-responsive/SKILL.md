# Tailwind Responsive Rules

Use mobile-first styling.
Add breakpoint prefixes only when needed.

Prefer:
- simple stacked mobile layout first
- progressive enhancement with `md:` and `lg:`
- clear width and max-width constraints
- readable wrapping behavior

Check:
- overflow
- text wrapping
- button stacking
- card grid collapse
- section padding at each breakpoint
